# RSS Агрегатор

Веб-приложение для добавления и просмотра RSS-потоков. Собрано на Vite, стили — Bootstrap 5.

## Ссылка на сайт

[Открыть приложение на Vercel](https://frontend-project-11-beta-two.vercel.app/) 

## Локальный запуск

```bash
npm install
npm run dev
```

Сборка для продакшена: `npm run build`. Превью: `npm run preview`.
