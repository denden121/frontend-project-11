# RSS Агрегатор

Веб-приложение для добавления и просмотра RSS-потоков. Собрано на Vite, стили — Bootstrap 5.

## Ссылка на сайт

[Открыть приложение на Vercel](https://your-app.vercel.app) — замените ссылку после деплоя.

## Локальный запуск

```bash
npm install
npm run dev
```

Сборка для продакшена: `npm run build`. Превью: `npm run preview`.
